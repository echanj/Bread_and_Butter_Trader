This Page Isn’t About Machine Learning...

There’s already plenty of content on machine learning across the internet. Here, we focus on documenting methods to utilize the data generated by the Trading Automaton — such as P&L data and Sharpe ratios — in an effort to improve its predictive capabilities. The goal is to keep it straightforward. The underlying principles here align with many of the core applications where machine learning shows promise. Perhaps your wondering if I have already embarked on a deep reinforcement learning project... NO !
