A "for dummies" Generalized Pairs Trading System (PairB&B)

Select an Output from a B&B Trading Automaton

The links below provide access to primary outputs from the trading automaton, each generated using different hyper-parameter configurations (labeled 1 through 4) with an input multi-stock data set. Each output corresponds to a specific historical training period, ending on November 1, 2024, which serves as the offset date.

The evaluation period begins on the next trading day and spans 20 trading days. The current outputs posted here are based on an evaluation date of Friday, November 8, 2024. Future updates on signals and analyses will be based on public interest.

The B&B Trading System includes two automaton sytems: an Instrument Pairs Trader and a Single Instrument Trader, each offering unique insights based on the screening of different trading strategies as a result of the hyper parameter setting combination used.

NB: Currently, links for extended analysis pages, which include more specific details of entry/exit tolerances as well as the ARIMA/GARCH models, have not been generated for all Trader runs. Those runs that have this supplementary information are indicated using "(links)."
